## Technologies
- Hugo framework [docs](https://gohugo.io/documentation/)

## How to use ?
- Edit Markdown files in `/content/` folder
- Edit theme design & layout in `themes/middleware`
- Edit theme configuration in `config.toml`

Open local server : 
- `hugo server -D`

Build static files:
- `hugo`

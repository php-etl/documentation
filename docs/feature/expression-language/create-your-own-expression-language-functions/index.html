Create you own Expression Language functions extending Symfony Expression Language

  • Introduction
  • Create a PHP project with composer
  • Install the symfony/expression-language dependency

Introduction

This package will extend the ExpressionLanguage component of Symfony to compile and evaluate your datas with custom functions.

The package is an example as a base tool to create you own functions that you will use in you Gyroscops projects.

As example, we will create a function to calculate the modulo

Create a PHP project with composer

Create you local project :

`mkdir calculator-expression-language && cd calculator-expression-language`

Start initialisation with composer :

`composer init`

Answer the question to have the base of a PHP project

Your composer.json file should be like this :


{
    "name": "my-project/calculator-expression-language",
    "type": "library",
    "autoload": {
        "psr-4": {
            "MyProject\\CalculatorExpressionLanguage\\": "src/"
        }
    },
    "require": {}
}

You should have this file schema :

src/
vendor/
   composer/
   psr/
   autoload.php
composer.json
composer.lock

Now, we have a base of a PHP project and we can start to develop the specific function

Create the specific Expression Language functions for Gyroscops

  1. Add PHP version required :

composer require php:^8.2

  1. Add the symfony/expression-language dependency

We need this dependency because the package we are developing is extending this Symfony package

composer require symfony/expression-language:*

  1. Create a Provider

We are creating a Provider to declare this function we will use in our Expression Language

Create the file src/MyProject/ExpressionLanguageProvider.php

<?php

namespace MyProject\Compoment\CalculatorExpressionLanguage;

use Symfony\Component\ExpressionLanguage\ExpressionFunction;
use Symfony\Component\ExpressionLanguage\ExpressionFunctionProviderInterface;

class CalculatorExpressionLanguageProvider implements ExpressionFunctionProviderInterface
{

    /**
     * @inheritDoc
     */
    public function getFunctions(): array
    {
        return [
            new GetRemainder('getRemainder'),
        ];
    }
}

  1. Create the class that will contain the code for the expression we want to add to Gyroscops

    src/GetRemainder.php

<?php

declare(strict_types=1);

namespace MyProject\Component\CalculatorExpressionLanguage;

use Symfony\Component\ExpressionLanguage\ExpressionFunction;

final class GetRemainder extends ExpressionFunction
{
    public function __construct($name)
    {
        parent::__construct(
            $name,
            $this->compile(...)->bindTo($this),
            $this->evaluate(...)->bindTo($this)
        );
    }

    private function compile(integer $value, integer $divider ): string
    {
        return <<<PHP
            (function () use (\$input) : integer {
                if(\$divisor === 0){
                    return false;
                }
            
                return \$divider % \$value;
            })()
            PHP;
    }

    private function evaluate(array $context, integer $value, integer $divisor ): integer
    {
        if($divisor === 0){
            return false;
        }
        return $value % $divisor;
    }
}
